Twenty years ago we had two applications to score the BAT (Bilingual Aphasia Test), the PCBAT and the MacBAT, neither of which run anymore. Now we have a web-based BAT scorer, that will run on any computer, on any mobile device, anywhere.

You can use this scorer to get a patient profile in 3 steps:

Step 1: the spreadsheets

Step 2: import the spreadsheets

Step 3: get the patient's profile

The BAT scorer is free, Open Source and developed using the GitHub project management tool, you can visit the project page to find out how you can help make it better.
